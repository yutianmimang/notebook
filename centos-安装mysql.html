Centos-安装mysql

2018年12月6日 0条评论 2次阅读 0人点赞 PENG YIFAN

添加REPO源

[root @ localhost~] #cd / usr / local / src /
[root @ localhost src] #wget http://repo.mysql.com/mysql57-community-release-el7-8.noarch.rpm

[root @ localhost src] #rpm -ivh mysql57-community-release-el7-8.noarch.rpm

[root @ localhost src] #yum -y install mysql-server

根据步骤安装就可以了，

默认配置文件路径：
配置文件：/etc/my.cnf中
日志文件：/var/log/var/log/mysqld.log
服务启动脚本：/usr/lib/systemd/system/mysqld.service
套接字文件：/ VAR /run/mysqld/mysqld.pid

配置my.cnf vim /etc/my.cnf

[mysqld]
##
删除前导＃并设置为
MySQL中 最重要的数据#cache的RAM量 。 从专用服务器的总RAM的70％开始，否则为10％。
#innodb_buffer_pool_size = 128M
##
删除前导＃以打开一个非常重要的数据完整性选项：
在备份之间 将 ＃changes 更改为二进制日志。
#log_bin
##
删除前导＃以设置主要用于报告服务器的选项。
＃服务器默认值对于事务和快速SELECT更快。
＃根据需要调整尺寸，试验以找到最佳值。 #join_buffer_size
= 128M
＃
sort_buffer_size = 2M ＃read_rnd_buffer_size = 2M
datadir = / var / lib / mysql
socket = / var / lib / mysql / mysql.sock
server_id = 1
expire_logs_days = 3

＃建议禁用符号链接以防止各种安全风险
symbolic-links = 0

log-error = / var / log / mysqld.log
pid-file = / var / run / mysqld / mysqld.pid

不过安装完成后，密码为随机密码，需要重置密码。

启动mysql的服务

service mysqld restart

重置密码

[root @ localhost~]  #grep“password”/var/log/mysqld.log 

输入mysql -u root -p密码进入

第一次登陆，需要重置密码要不什么也不能操作

最后记得刷新权限;

刷新权限

也可以直接再添加新用户

创建用户'root'@'％'由'您的密码'识别;
